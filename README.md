# 📜 Text Generation with Markov Chains

## 🎯 Objective
The goal of this project is to implement a **Markov Chain-based text generator** that learns from a given dataset and generates new, contextually relevant text. This helps understand the basics of probabilistic modeling and sequence generation.

---

## 🛠️ Tools Used
- **Python 3** – Core programming language
- **Jupyter Notebook** – For interactive code development
- **NumPy** – For handling numerical operations
- **Collections** – For managing frequency counts and state transitions
- **Random** – For probabilistic word selection

---

## 💡 Key Highlights
- Implementation of **Markov Chains** from scratch.
- Ability to train on any text corpus.
- Generates new text that mimics the style of the training data.
- Customizable chain length for better context retention.
- Clear code structure for easy learning and modification.
